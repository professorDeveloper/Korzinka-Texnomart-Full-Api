## Run Locally

Clone the project

```bash
  git clone https://github.com/professorDeveloper/Korzinka-Texnomart-Full-Api
```

Go to the project directory

```bash
  cd Korzinka-Texnomart-Full-Api
```

Install dependencies

```bash
  flutter pub get
```

Start the Application

```bash
  flutter run
```
